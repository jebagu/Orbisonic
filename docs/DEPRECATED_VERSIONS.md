# Deprecated Orbisonic Versions

`main` is the canonical Orbisonic branch.

The current winner was promoted from `pure-audio-branch-2`. Older branches and release tags are preserved for history, but they are no longer the active development or launch targets.

Root-level launchers for older branches and releases now print a deprecation notice and open `main` instead of launching the old ref.
