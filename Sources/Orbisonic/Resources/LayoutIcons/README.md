# Orbisonic layout icons

Transparent, tintable SVG source icons copied from the layout menu screenshot.

Use them as single-color template art. Each SVG uses `currentColor` for dots,
strokes, and checkmarks so selected, inactive, and disabled states can be
controlled by the host UI.
